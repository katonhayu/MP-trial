**Customer Journey (8 pages):**

- **Homepage** (`/`) - Hero section, featured products, stats, and CTA
- **Products List** (`/products`) - Browse with search, filters, and sorting
- **Product Detail** (`/products/[id]`) - Full product info, reviews, related items
- **Cart** (`/cart`) - Shopping cart with item management
- **Checkout** (`/checkout`) - Secure payment form
- **Success** (`/checkout/success`) - Order confirmation with downloads
- **Login** (`/auth/login`) - Authentication with social login options
- **Register** (`/auth/register`) - Account creation for buyers/sellers
- **My Purchases** (`/customer/purchases`) - Download history


**Seller Journey (6 pages):**

- **Dashboard** (`/seller/dashboard`) - Revenue charts, stats, recent orders
- **Products** (`/seller/products`) - Product listing table with actions
- **Add Product** (`/seller/products/new`) - Full product creation form
- **Orders** (`/seller/orders`) - Order management with filters
- **Analytics** (`/seller/analytics`) - Detailed charts and performance metrics
- **Settings** (`/seller/settings`) - Profile, store, payouts, notifications


**System Flow** (`/system-flow`) - Visual documentation of both customer and seller journeys with clickable navigation to all pages.
